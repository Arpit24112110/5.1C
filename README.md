# SIT708 Task 5.1 – Android Apps

## Overview
This repository contains two Android applications developed as part of SIT708 Task 5.1.

---

## 1. Sports News Feed App

### Features
- Single Activity with multiple Fragments
- Featured matches (horizontal RecyclerView)
- Latest news (vertical RecyclerView)
- Detail screen for news articles
- Related stories section
- Search/filter by sport category
- Bookmark functionality
- Bookmarks screen

---

## 2. iStream App

### Features
- User Login and Signup
- Room Database integration
- YouTube video playback using embedded WebView (IFrame API)
- Add videos to personal playlist
- View playlist
- Click playlist item to load video in player
- Logout functionality
- Input validation and error handling

---

## Technologies Used
- Kotlin
- Android Studio
- RecyclerView
- Room Database
- WebView (YouTube IFrame API)
